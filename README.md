# E-Commerce Sales Analytics System (SQL Project)

## Project Overview
This project demonstrates database design and sales analytics using MySQL.

## Database Features
- User management
- Product catalog
- Order tracking
- Payment analysis

## Technologies Used
- MySQL
- SQL (JOIN, GROUP BY, Aggregations)

## Key Analysis Performed
- Total revenue calculation
- Top selling products
- Category-wise revenue
- High-value customers
- Payment method usage analysis

## How to Run
1. Open MySQL Workbench
2. Run ecommerce_sales_analytics.sql
3. Execute analytics queries
